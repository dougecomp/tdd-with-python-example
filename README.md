# tdd-with-python-example

Basic Example of TDD with python posted on Substack: https://open.substack.com/pub/dougecomp/p/tdd-desenvolvendo-com-testes-em-primeiro

## Install poetry

```sh
pip install poetry
```

## Install dependencies
```sh
poetry install
```

## Run tests

Pytest
```sh
pytest
```

OU

Pytest-watch
```sh
ptw
```
